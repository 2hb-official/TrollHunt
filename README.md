# TrollHunt
Rendszerváltást követő HSP végleszámolás a trollokkal

## Telepítési Útmutató

A TrollHunt egy egyszerű böngésző alapú játék, amelyhez nincs szükség szerverre vagy bonyolult beállításokra. Az alábbi lépéseket követve könnyedén elindíthatod:

1.  **Töltsd le a játékot:**
    *   Navigálj a játék GitHub repository-jához (ha van, vagy ahonnan letöltötted).
    *   Keresd meg a **"Code"** gombot (általában zöld színű).
    *   Kattints rá, majd válaszd a **"Download ZIP"** opciót. Ez letölti a teljes projektet egy tömörített `.zip` fájlként.

2.  **Csomagold ki a fájlokat:**
    *   Keresd meg a letöltött `.zip` fájlt a számítógépeden (általában a "Letöltések" mappában található).
    *   Kattints jobb gombbal a fájlra, majd válaszd a **"Kibontás ide"** vagy **"Extract All"** opciót. Ez létrehoz egy új mappát a játék fájljaival.

3.  **Indítsd el a játékot:**
    *   Nyisd meg a kicsomagolt mappát.
    *   Keresd meg az **`index.html`** nevű fájlt.
    *   Kattints duplán az `index.html` fájlra. Ez automatikusan megnyitja a játékot az alapértelmezett böngésződben (pl. Chrome, Firefox, Edge).

---

**Fontos megjegyzés:** A játék futtatásához **nincs szükség semmilyen webkiszolgálóra** (pl. Apache, Nginx). Közvetlenül a fájlrendszerről is futtatható.
